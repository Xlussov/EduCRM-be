# EduCRM Backend in Go

Backend system for an **educational center** to manage branches, teachers, students, lesson schedules, attendance, and subscription plans.

---

## ⚙️ Configuration

* Config files are located in the `config/` folder (`config.yml`)
* Environment variables may override settings (e.g., `APP_ENV=local|dev|prod`)

---

## 🚀 Running the Application

```bash
go run ./cmd/app/main.go
```

Server will start at `http://localhost:8080`.
